Поиск документов на cbr.ru
Реализовать программу, которая ищет документы pdf на сайте https://www.cbr.ru.
Документы искать рекурсивно в разделах, доступных в левом меню сайта.
Предусмотреть возможность запуска с разными браузерами (chrome, firefox).
Для поиска элементов использовать XPATH.
Модифицировать программу для работы без браузера. Например используя модуль requests.
Модифицировать программу для работы в несколько потоков или асинхронных задач.
